Proyecto Base de Datos Empresa - SQL

Descripción
Este proyecto simula la base de datos de una empresa utilizando MySQL.  
Incluye tablas de empleados, clientes, productos y ventas.

El objetivo es practicar consultas SQL aplicadas a un entorno empresarial real.

Tecnologías utilizadas
- MySQL
- SQL

Conceptos practicados
- SELECT
- DISTINCT
- WHERE
- AND / OR / NOT
- IN / NOT IN
- LIKE
- ORDER BY
- LIMIT
- Funciones de agregado (SUM, COUNT, AVG)
- GROUP BY
- HAVING
- Claves foráneas (FOREIGN KEY)

Objetivo del proyecto
Practicar consultas SQL similares a las que pueden solicitarse en entrevistas técnicas para puestos de Analista de Datos o desarrollador junior.
